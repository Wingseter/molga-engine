# Project Overview
**Name**: molga-engine
**Type**: 2D Game Engine (C++)

A custom 2D game engine featuring an editor and a runtime environment. It supports entity-component-system (ECS) architecture, scripting, physics (box collider), and audio.

## Key Components
- **Editor**: Built with ImGui, allows scene editing, hierarchy management, and property inspection.
- **Runtime**: A standalone player for running games created with the engine.
- **ECS**: Component-based architecture for game objects.

## Entry Points
- Editor: `src/main.cpp`
- Runtime: `src/runtime_main.cpp`
